Prerequisites & Requirements

Technical Requirements

  • Laptop with Linux/macOS/Windows
  • 8GB+ RAM recommended
  • USB 3.0 port for SDR devices
  • Python 3.8+ installed
  • GNURadio (optional but recommended)

Knowledge Prerequisites

  • Basic understanding of RF concepts
  • Python programming experience
  • Familiarity with command line tools
  • Basic networking knowledge
  • Interest in drone technology

Software Setup

  • Wireshark for packet analysis
  • GQRX or SDR# for RF analysis
  • ArduPilot SITL simulator
  • MAVProxy for MAVLink testing
  • Text editor (VS Code recommended)
