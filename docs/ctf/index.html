Drone Architecture

Diagram: RC link → Flight controller → Companion/GCS. FPV transmitter (analog/digital) is separate and often in the 5.7–5.9 GHz band.

How to use this handout

  1. Download the sample files from /ctf/samples/ — these are safe SITL & IQ samples.
  2. Follow each lab's README in /ctf/labs/. Each README contains reproducible commands for GNURadio, pymavlink, and visualization.
  3. Use the supplied GNURadio flowgraph /ctf/grc/lab2_flowgraph.grc to demod analog FPV.
  4. Run the example scripts in /ctf/scripts/ on your laptop. The scripts are documented and meant for SITL/offline data only.
