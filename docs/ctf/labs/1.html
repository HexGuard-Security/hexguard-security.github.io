Lab 1 Tello Recon

discover connect and command

objective

discover the drone network identify services and send basic commands using udp

requirements

  • laptop with python 3
  • wifi capable of connecting to 2.4 ghz ssid
  • dji tello drone
  • nmap wireshark or tshark

steps

connect

nmcli dev wifi list
nmcli dev wifi connect "TELLO-XXXXXX"

verify

ping 192.168.10.1
nmap -sV -Pn 192.168.10.1
sudo tshark -i wlan0 -Y "ip.addr == 192.168.10.1"

python control script

import socket

TELLO_IP = "192.168.10.1"
TELLO_PORT = 8889
LOCAL_PORT = 9000

sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
sock.bind(("", LOCAL_PORT))

def send(cmd):
    sock.sendto(cmd.encode("utf-8"), (TELLO_IP, TELLO_PORT))
    resp, _ = sock.recvfrom(1024)
    print(cmd, resp.decode("utf-8"))

send("command")
send("takeoff")
send("land")

run

python3 tello_control.py
