Lab 2 Analog FPV Demod

decode composite video from iq file using gnuradio

objective

demodulate an analog fpv iq capture produce a playable composite file

requirements

  • gnuradio 3.10 or newer
  • gr osmosdr or rtl sdr drivers
  • ffmpeg or ffplay for playback
  • sample iq file fpv_sample f32

flowgraph summary

  1. file source read iq file
  2. frequency translating fir filter shift carrier to baseband
  3. low pass filter isolate video fm
  4. quadrature demod fm demodulation
  5. rational resampler set composite sample rate
  6. file sink write raw composite

record example with hackrf

hackrf_transfer -r fpv_sample.f32 -f 5740000000 -s 2000000 -n 2000000

run gnuradio companion open lab2_flowgraph grc

gnuradio-companion grc/lab2_flowgraph.grc

play composite output

ffplay -f rawvideo -pixel_format gray16 -video_size 720x480 composite_out.raw
