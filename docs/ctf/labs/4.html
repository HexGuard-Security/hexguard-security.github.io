Lab 4 MAVLink Capture

capture mavlink from sitl and inspect with wireshark and pymavlink

objective

produce a pcap of mavlink traffic parse messages and export csv

requirements

  • ardupilot sitl installed
  • tcpdump wireshark
  • pymavlink python library

steps

start sitl

cd ~/ardupilot
Tools/autotest/sim_vehicle.py -v ArduCopter -f quad --console --map

capture to pcap

sudo tcpdump -i lo -w sitl_mav.pcap 'tcp port 5760'

parse with pymavlink

from pymavlink import mavutil
m = mavutil.mavlink_connection('sitl_mav.pcap', dialect='ardupilotmega')
while True:
    msg = m.recv_match(blocking=False)
    if msg is None:
        break
    if msg.get_type() == 'GLOBAL_POSITION_INT':
        print(msg.time_boot_ms, msg.lat, msg.lon, msg.alt)
